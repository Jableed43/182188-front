Anatomía de una regla CSS

    selector {
        propiedad: valor;
    }

Selector: ¿a quién le hablo? · Propiedad: ¿qué le cambio? · Valor: ¿cómo lo quiero?

Este texto usa CSS inline (style="..."). Ensucia el HTML: no lo hagas.

Esta caja usa CSS interno (una regla dentro de <style> en el head).

La forma correcta y profesional es el CSS externo: un archivo .css enlazado con <link rel="stylesheet" href="css/styles.css">, como se ve en index.html.
